    Classe para exibição de texto em ambiente Shell (modo texto) A classe Text4Shell possibilita ao desenvolvedor uma maneira fácil e elegante de exibir conteúdo em ambiente Shell.

    Se você utiliza o PHP CLI (PHP Command Line Interface), Text4Shell é a solução ideal para exibir conteúdo.

    Cria uma tabela com os dados recebidos.

    table(array $data, array $colmodel, string $title) : string

    method Cria uma tabela os dados contidos em $data. $data precisa estar no formato:
    since 1.0

    Parameters

    $data

    array

    Um array multidimensional com os dados de linhas e colunas par exibir.

    $colmodel

    array

    Um array especificando os rótulos das colunas, a largura de cada coluna e o alinhamento de cada coluna. Observe que a soma das larguras das colunas deve ser igual ao valor de Text4Shell::cols. Se um rótulo não for especificado, o id da coluna será utilizado. Os alinhamentos válidos são LEFT, RIGHT e CENTER (em maiúsculas ou minúsculas). Se nenhum for especificado, LEFT é adotado. O array segue a estrutura: array( 'campo1' => array('label' => 'Rótulo', 'len' => 10, 'align' => 'LEFT'), 'campo2' => array([...]), [...])

    $title

    string

    [Opcional] Se definido, mostra o texto com título da tabela.
